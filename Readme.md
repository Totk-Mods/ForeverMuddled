# Usage

## Windows

```powershell
git clone "https://github.com/Totk-Mods/ForeverMuddled";
cd "ForeverMuddled";
.\muddle;
```

## Unix (Linux/MacOS)

```sh
git clone "https://github.com/Totk-Mods/ForeverMuddled" && \
cd "ForeverMuddled" && \
./muddle.sh
```
